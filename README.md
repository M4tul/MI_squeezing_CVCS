# MI_squeezing_CVCS
Repository to store data of research published under https://doi.org/10.1103/78tf-57jf. 

We present the raw numpy data and a simple Jupyter notebook that recreates the figures presented in https://doi.org/10.1103/78tf-57jf.
Data are stored in data.zip in the format of numpy arrays.
Jupyter notebook figures_MI_squeezing.ipynb plots all the data.
